# META 商家驗證 - 解決方案

## ❌ 問題說明

META 顯示錯誤：
> 「你無法使用一般網站網域（例如：www.yahoo.com、www.gmail.com、www.hotmail.com）驗證你的企業商家。請提供有效的商家網站。」

**原因分析**：
META 認為您的網站缺少必要的商家資訊，不符合商家網站的要求。

---

**目前狀況**：
- ✅ 有隱私權政策頁面：`/privacy`

**解決方法**：需要創建「關於我們」頁面

- 商家名稱

## 🎯 快速解決方案（臨時）

如果急需通過驗證，可以：

### 方案 1：在現有頁面添加商家資訊

**在隱私權政策或服務條款頁面底部添加**：

```html
<div>
  <h2>關於我們</h2>
  <p><strong>商家名稱</strong>：ChaosRegistry</p>
  <p><strong>服務內容</strong>：ChaosRegistry 是一款以虛構、趣味話題為核心的娛樂性投票平台。使用者可瀏覽主題、以代幣投票、發起投票，或使用代幣提高主題曝光度。</p>
  <p><strong>聯絡方式</strong>：請透過聯絡我們頁面與我們聯繫</p>
</div>
```

### 方案 2：使用 Vercel 的公開頁面

1. **在 Vercel 專案中**：
   - 創建一個 `public/about.html` 檔案
   - 添加商家資訊
   - 部署後可以通過 `https://chaos-registry.vercel.app/about.html` 訪問

2. **內容範例**：
```html
<!DOCTYPE html>
<html>
<head>
  <title>關於我們 - ChaosRegistry</title>
  <meta charset="UTF-8">
</head>
<body>
  <h1>關於 ChaosRegistry</h1>
  <p>ChaosRegistry 是一款以虛構、趣味話題為核心的娛樂性投票平台。</p>
  <p>服務內容：使用者可瀏覽主題、以代幣投票、發起投票，或使用代幣提高主題曝光度。</p>
  <p>聯絡方式：請透過 <a href="/contact">聯絡我們</a> 頁面與我們聯繫</p>
  <p><a href="/privacy">隱私權政策</a> | <a href="/terms">服務條款</a></p>
</body>
</html>
```
