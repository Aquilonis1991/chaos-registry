# 🌐 VoteChaos Web 版本使用說明

## ✅ 開發伺服器已啟動

執行 `npm run dev` 後，開發伺服器正在啟動...

## 📱 如何訪問

### 在電腦上
1. 等待終端機顯示網址（通常是 `http://localhost:5173`）
2. 在瀏覽器中打開該網址
3. 所有功能都可以正常使用

### 在手機上（同網路）
1. 終端機會顯示網路地址，例如：`http://192.168.x.x:5173`
2. 在手機瀏覽器中打開該網址
3. 可以體驗接近 APP 的效果

### 如果想繼續解決
最快的方式仍然是取得 Logcat 日誌：

```powershell
# 5 分鐘設定
adb devices
adb logcat -c
adb logcat VoteChaos:D AndroidRuntime:E *:F > crash.log
# 開啟 APP（崩潰）
# 按 Ctrl+C
# 提供 crash.log
```

有了日誌，我可以立即定位並修復問題。

### 替代方案
1. 暫時專注 Web 版本開發
2. 等有其他 Android 設備時再測試
3. 考慮使用 React Native 或 Flutter 重寫
