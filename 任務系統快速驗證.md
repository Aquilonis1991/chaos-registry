# ⚡ 任務系統快速驗證（5 分鐘版本）

## 🚀 快速測試步驟

### 1️⃣ 開啟 Console 並訪問任務頁
- 按 `F12` 開啟開發者工具
- 切換到 **Console** 標籤
- 訪問任務頁面 (`/mission`)
- **確認**：看到 `📊 User Stats:` 日誌

### 2️⃣ 測試「新手上路」
- [ ] 查看任務顯示：`0% 完成`
- [ ] 去首頁對任一主題投票（免費或代幣）
- [ ] 返回任務頁
- [ ] **確認**：顯示 `100% 完成` ✓

### 3️⃣ 測試「話題創造者」
- [ ] 查看任務顯示：`0% 完成`
- [ ] 建立一個新主題（點擊底部「+」）
- [ ] 返回任務頁
- [ ] **確認**：顯示 `100% 完成` ✓

### 4️⃣ 測試「投票愛好者」
- [ ] 查看當前進度（例如：`30% 完成`）
- [ ] 對 **10 個不同主題** 投票（每次對不同主題）
- [ ] 每投票一次，檢查進度增加：10% → 20% → 30% → ...
- [ ] 投票第 10 個不同主題後
- [ ] **確認**：顯示 `100% 完成` ✓

## 🔍 Console 檢查點

每次訪問任務頁應該看到：

```
✅ Votes fetched: X votes
✅ Free votes fetched: X free votes  
✅ Topics created: X
📊 User Stats: {
  totalVotes: X,           ← 用於「新手上路」
  uniqueTopicVotes: X,     ← 用於「投票愛好者」 
  topicsCreated: X         ← 用於「話題創造者」
}
```

## ❌ 如果出問題

1. **刷新頁面**（`Ctrl+F5`）
2. **檢查 Console 錯誤**
3. **確認數據表中有記錄**（見下方 SQL）

---

## 🔧 快速 SQL 檢查

```sql
-- 替換 <你的 user_id>
SELECT 
  (SELECT COUNT(*) FROM votes WHERE user_id = '<你的 user_id>') AS 代幣投票數,
  (SELECT COUNT(*) FROM free_votes WHERE user_id = '<你的 user_id>') AS 免費投票數,
  (SELECT COUNT(*) FROM topics WHERE creator_id = '<你的 user_id>') AS 創建主題數;
```
