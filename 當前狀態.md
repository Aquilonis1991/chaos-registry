## ❌ 缺少項目（10%）

### Java JDK（必需）

**錯誤訊息**：
```
ERROR: JAVA_HOME is not set and no 'java' command could be found in your PATH.
```

**原因**：
建置 Android APK 需要 Java 開發工具包（JDK）來編譯 Gradle 專案。

## 🎯 立即行動：安裝 JDK

### 🌟 推薦：下載 Android Studio（一勞永逸）

**為什麼選這個？**
- ✅ 包含所有需要的工具（JDK、Android SDK、Gradle）
- ✅ 未來可用於調試和開發
- ✅ 官方推薦方式
- ✅ 圖形化介面管理

**下載連結**：
🔗 https://developer.android.com/studio

**步驟**：
1. 點擊「Download Android Studio」
2. 執行安裝程式
3. 選擇「Standard」安裝
4. 等待完成（30-60 分鐘）
5. **重新開啟 PowerShell**

---

### ⚡ 替代方案：只安裝 JDK 17（快速）

**為什麼選這個？**
- ⚡ 安裝快速（10 分鐘）
- 💾 佔用空間小（~500 MB）
- ✅ 足夠建置 APK

**下載連結**：
🔗 https://adoptium.net/temurin/releases/?version=17

**步驟**：
1. 選擇：
   - Operating System: **Windows**
   - Architecture: **x64**
   - Package Type: **JDK**
   - Version: **17 (LTS)**
2. 下載 `.msi` 檔案
3. 執行安裝
4. ✅ **勾選** "Set JAVA_HOME variable"
5. ✅ **勾選** "Add to PATH"
6. **重新開啟 PowerShell**

## 📋 安裝完成後的操作

### 步驟 1: 驗證 JDK 安裝

**重新開啟 PowerShell**，然後執行：

```powershell
java -version
```

**應該看到**：
```
openjdk version "17.0.x" 2024-xx-xx
...
```

### 步驟 2: 檢查 JAVA_HOME

```powershell
echo $env:JAVA_HOME
```

**應該顯示**：
```
C:\Program Files\Eclipse Adoptium\jdk-17.x.x-hotspot
```
或類似路徑。

### 步驟 3: 回到專案並建置

```powershell
cd "C:\Users\USER\Documents\工作用\votechaos-main"
cd android
.\gradlew.bat assembleDebug
```

**或者使用腳本**：
```powershell
.\build-now.ps1
```
